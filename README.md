Sou desenvolvedor **Full Stack**, com foco em aplicações **mobile** (React Native + Expo) e **web**. Gosto de construir produtos de ponta a ponta — do backend à interface — e estou sempre aprendendo tecnologias novas através de projetos reais.

- 🔭 Atualmente trabalhando em projetos com **React Native, Expo, Firebase e React**
